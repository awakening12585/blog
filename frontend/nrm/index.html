# nrm

nrm是一个方便的npm的镜像仓库管理工具，可以便捷的在各个镜像仓库之间切换。

# 安装

npm install -g nrm --registry=https://registry.npm.taobao.org
//从淘宝镜像下全局安装nrm

2

# 命令

//展示nrm镜像
nrm ls   
//使用淘宝镜像
nrm use taobao
